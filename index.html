Словарь для записи японских слов

Я немного увлекаюсь японским, поэтому решил сделать такое приложение, куда бы мог записывать переводы чего-либо. Можно как и для обычных слов использовать так и для песен и прочего.

Приложение делалось для личного пользования, поэтому я сильно не заморачивался. Может когда-нибудь переделаю.
P.S. На видео почему то select не отображается. Там выбор между разными коллекциями.
